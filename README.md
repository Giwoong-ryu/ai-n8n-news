# -# AI 뉴스 자동화 워크플로우 교육

n8n을 활용한 AI 뉴스 자동 요약 및 카카오톡 전송 워크플로우 교육 자료입니다.

## 📚 교육 자료

### 1. 강의 자료
- [n8n 교육자료 _ 크롤링부터 카톡전송 까지.xlsx](./AI뉴스_워크플로우_강의자료.xlsx) - 전체 워크플로우 가이드

### 2. 워크플로우 구조
```
RSS Read → Limit → Code → Gemini → Kakao
```

### 3. 필요한 준비물
- n8n 설치 (Docker 또는 Self-hosted)
- Google Gemini API 키
- 카카오 디벨로퍼스 앱 등록

### 4. 주요 기능
- ✅ RSS 피드 자동 수집
- ✅ AI(Gemini)로 10개 중 3개 선별 + 요약
- ✅ 카카오톡 자동 전송
- ✅ 완전 무료 운영
